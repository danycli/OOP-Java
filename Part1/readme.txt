============================================================
                 PART 1: TRIANGLE PROGRAM
============================================================

>> OVERVIEW <<
------------------------------------------------------------
This program implements the "Triangle" class as described in 
the assignment sheet. It demonstrates key Object-Oriented 
Programming (OOP) concepts such as constructors, methods, 
static members, and object comparison.


>> MAIN FEATURES <<
------------------------------------------------------------
• Implementation of the Triangle Class
  - Includes data members for side lengths, area, perimeter, 
    and triangle type.

• Constructors
  - Three-argument constructor initializes triangle sides.
  - Copy constructor creates a new triangle from an existing one.

• Computation Methods
  - Calculates the area using Heron’s Formula.
  - Computes the perimeter by summing all sides.

• Type Determination
  - Identifies whether the triangle is:
      → Equilateral  (all sides equal)
      → Isosceles    (two sides equal)
      → Scalene      (all sides different)

• Display Method
  - Displays complete triangle information such as sides, 
    perimeter, area, and type.

• Static Method
  - Compares the types of three triangle objects to determine 
    if they are of the same or different types.


>> LEARNING OBJECTIVES <<
------------------------------------------------------------
• Understand the use of constructors (parameterized & copy).
• Practice method creation and object manipulation.
• Apply conditional logic for type classification.
• Demonstrate static method usage for object comparison.
